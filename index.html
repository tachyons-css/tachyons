About

A set of single purpose css classes that make it easy to design fully responsive websites/apps in the browser without writing any css.

Build jank-free highly performant interfaces with just 7.6kb of css

Single-purpose float classes

The CSS


.fl { float: left; }
.fr { float: right; }
.fn { float: none; }

The HTML


<div class="cf">
  <div class="fl pal bg-orange"> Float left  </div>
  <div class="fr pal bg-aqua">   Float right </div>
  <div class="fn pal bg-red tc"> Float none  </div>
</div>
